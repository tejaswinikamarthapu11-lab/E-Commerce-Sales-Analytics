# 📊 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

This project analyzes e-commerce sales data to uncover customer behavior, product performance, sales trends, and business insights. The analysis was performed using Python for data cleaning and transformation, and Power BI for interactive dashboard creation.

The project helps stakeholders monitor key business metrics, identify high-performing products, understand customer purchasing patterns, and support data-driven decision-making.



## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand customer purchasing behavior
* Track revenue and profit trends
* Create interactive dashboards for business insights
* Support strategic decision-making through data visualization

## 🔍 Data Analysis Process

### 1. Data Collection

Collected e-commerce transaction data containing sales, customers, products, orders, and revenue information.

### 2. Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized column formats
* Corrected data inconsistencies

### 3. Data Transformation

* Created calculated metrics
* Generated KPIs
* Performed feature engineering
* Prepared data for visualization

### 4. Dashboard Development

Built interactive Power BI dashboards to monitor:

* Sales Performance
* Customer Insights
* Product Performance
* Revenue Trends
* Regional Analysis
