# 📊 E-Commerce Sales Analytics Dashboard

## Project Overview

This project focuses on analyzing e-commerce sales data to uncover valuable business insights related to customer behavior, product performance, sales trends, and profitability.

Using Python for data cleaning and transformation and Power BI for visualization, interactive dashboards were developed to support data-driven decision-making.

---

## 🎯 Business Objectives

* Analyze sales performance across products and categories
* Identify top-performing products
* Understand customer purchasing behavior
* Monitor revenue and profit trends
* Build interactive dashboards for stakeholders
